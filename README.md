# Домашний бюджет

Приложение для учёта домашних расходов и доходов. Часть проекта [SeniorHub — Офис пенсионера](https://github.com/andrey1b).

## Возможности

- Учёт **расходов** и **доходов** по счетам и категориям
- **Переносы** между счетами
- **Импорт фискальных чеков** по QR-коду с телефона (Android + iPhone)
- **Импорт/экспорт** данных из Excel (.xlsx)
- Отчёты и итоговая строка по каждой таблице
- Поддержка языков: русский, украинский, английский

## Установка

**[⬇ Скачать последнюю версию](https://github.com/andrey1b/HomeAccounting/releases/latest)**

1. Нажмите ссылку выше и скачайте `HomeAccounting_Setup_x.y.z.exe`
2. Запустите от имени **администратора**
3. При запросе брандмауэра — нажмите **«Разрешить доступ»**

## Приём чеков с телефона

| Платформа | Способ |
|---|---|
| **Android** | Установите ComfortBuh (QR-код в правой панели программы) |
| **iPhone** | Откройте ссылку из правой панели в Safari → вставьте URL чека |

## Требования

- Windows 10 / 11 (x64)
- Права администратора при установке (для правила брандмауэра)

## Сборка из исходников

```
dotnet publish HomeAccounting.csproj -c Release -r win-x64 --self-contained -p:PublishSingleFile=true -o publish
```
